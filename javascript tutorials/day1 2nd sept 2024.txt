1.what is dom?dom tree?
A:The Document Object Model (DOM) is a tree-like structure that represents the structure of a document in memory and connects web pages to programming languages.The DOM tree is a hierarchical representation of an HTML or XML document, and each node in the tree represents a different element of the document.
The W3C DOM standard is separated into 3 different parts:

Core DOM - standard model for all document types
XML DOM - standard model for XML documents
HTML DOM - standard model for HTML documents
